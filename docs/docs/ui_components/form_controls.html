Form Controls

Doodle has many built-in controls that cover a range of data types for forms. These controls come as functions that return FieldVisualizers. They are all found within the io.nacular.doodle.controls.form package.

tip

Usage of these controls still require explicit installation of Behaviors. Those shown below use behaviors from the BasicTheme or NativeTheme.

Custom Controlsâ€‹

Doodle makes it easy to create custom form controls. You simply implement FieldVisualizer or use the field DSL. Each control is a simple binding between a Field, its initial value and a View.

Here is an example of how you might add a boolean field with some text.

fun switchField(text: StyledText) = field<Boolean> {
    container {
        focusable = false // ensure wrapping container isn't focusable
        this += Label(text).apply {
            fitText             = setOf(Height)
            horizontalAlignment = Left
        }
        this += Switch().apply {
            initial.ifValid { selected = it } // adopt initial value if present

            selectedChanged += { _,_,_ ->
                state = Valid(selected) // update field as switch changes
            }

            size  = Size(40, 25)
            state = Valid(selected) // ensure field is valid at beginning
        }

        layout = constrain(children[0], children[1]) { label, switch ->
            switch.right   = parent.right - 10
            switch.centerY = parent.centerY

            label.left     = parent.left + 10
            label.right    = switch.left - 10
            label.centerY  = switch.centerY
        }
    }
}

fun colorStrip(first: Color, second: Color, vararg rest: Color, size: Double = 30.0) = field<Color> {
    container {
        focusable = false // ensure wrapping container isn't focusable
        val buttonGroup = ButtonGroup()
        this += listOf(first, second, *rest).map { color ->
            // use toggle button since it has desirable selection behavior
            ToggleButton().apply {
                this.size = Size(size)
                behavior = simpleButtonRenderer { button, canvas ->
                    val radius    = width * 0.2
                    val thickness = 2.0
                    val fillColor = if (button.model.pointerOver) color.lighter() else color

                    when {
                        selected -> canvas.rect(bounds.atOrigin.inset(thickness / 2), radius = radius, color = fillColor, stroke = Stroke(color.inverted, thickness = thickness))
                        else     -> canvas.rect(bounds.atOrigin, radius = radius, color = fillColor)
                    }
                }

                acceptsThemes = false

                selectedChanged += { _,_,_ ->
                    if (selected) {
                        state = Valid(color) // update field state when selected
                    }
                }

                initial.ifValid {
                    selected = it == color // adopt initial value if it matches something in the list
                }

                buttonGroup += this // ensure only 1 item is ever selected
            }
        }

        layout = HorizontalFlowLayout(spacing = 4.0)
    }
}
Form { this(
    Green to colorStrip(Red, Green, Blue, BlueColor, Black),
    onInvalid = {}
) { color ->
    println("Form valid: $color")
}
